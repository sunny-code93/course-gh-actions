# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it by opening an issue or contacting the maintainers directly. Do **not** disclose the vulnerability publicly until it has been reviewed and addressed.

- Please provide as much detail as possible about the vulnerability and steps to reproduce it.
- We will acknowledge your report within 5 business days and keep you informed of our progress.

## Supported Versions

We currently support the latest version of this project. Older versions may not receive security updates.

## Security Best Practices

- Do not share sensitive information (such as credentials or secrets) in issues, pull requests, or code.
- Use environment variables or secret management tools for sensitive data.
- Keep dependencies up to date and monitor for known vulnerabilities.

## Responsible Disclosure

We appreciate responsible disclosure of security issues. We will investigate all reports and address them as quickly as possible.

Thank you for helping keep this project and its users safe!
